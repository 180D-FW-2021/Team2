# IMU Gestures 

## SSH into raspberry pi 
```
ssh pi@raspberrypi.local
```

## Install all dependencies on raspberry pi 
```
pip install paho-mqtt
```

## Run the berryIMU.py file 
```
python berryIMY.py
```
