## Controls

### Keyboard Controls

- Forward/back/left/right: arrow keys/ WASD
- Rotate camera: 'z' (left) 'x' (right)
- Jump: space bar
- Duck: 'p'
- Pause Menu: Esc

### Voice Commands

- "Stop": open pause menu
- "Start": resume game

### Web Cam

- Jump
- Duck
